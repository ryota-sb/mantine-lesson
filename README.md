# Next.js + Supabase + Mantine

- Next.js (frontend)
- Supabase (backend)
- Mantine (UI component)

# Next.js

```bash
yarn create next-app --typescript
```

# Supabase

```bash
yarn add @supabase/supabase-js
```

# Mantine

```bash
yarn add @mantine/core @mantine/hooks @mantine/dates @mantine/next @mantine/
forms
```
